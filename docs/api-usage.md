# API Usage

## Microsoft Graph
- `/me/messages`
- `/me/mailFolders/inbox/messages`
- `/me/events`
- `/me/calendarView`
- `/me/todo/lists` and tasks endpoints

## JobTread (read-focused)
- Job list/status
- Open tasks
- Financial snapshot fields used for prioritization

## Usage Principles
- Minimize calls
- Cache where appropriate
- Do not request broad scopes without clear feature need
